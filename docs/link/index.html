オープントリガーがリンクのメガメニュー

実装ルール

  • 「当社について」や「お知らせ」など、メニューオープンのトリガーになる要素がaタグ
  • PCではホバーでメニューを展開
  • タッチデバイスと768px未満では、「∨」をタップでメニューを展開し、「当社について」や「お知らせ」をタップでリンク先へ遷移
  • 開いているメニューは1つまで
  • キーボード操作でもメニューを展開可能に
  • メニューが開いた状態で768pxを跨いだら、メニューを閉じる
